# Archive

Historical artifacts kept for reference only — **not used by the app**.

- **`project/`** — the original Claude Design prototype bundle
  (`MSBP Project Manager.dc.html`, `support.js`) plus its raw exported data
  (`data/`) and source uploads (`uploads/`, including the original
  `MSBP Project Consolidated (1).xlsx` and design screenshots).
- **`chat-history/`** — chat transcripts from building the prototype.

The live app reads its seed from `public/data/projects.json`. The downloadable
source spreadsheet served by the app lives at `public/data/source.xlsx`.
